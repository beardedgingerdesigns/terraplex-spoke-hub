# Terminology Rules

> Word-level rules for dealer site copy. These are non-negotiable — use the preferred terms verbatim.

- Say "aerial application" — NEVER "crop dusting" (industry considers it derogatory)
- Say "Authorized Terraplex Dealer" — NOT "Authorized Revolution Drone Dealer"
- Say "GTEEX Revolution" for the product brand, "Terraplex" for the dealer network (see [research/manufacturers/terraplex.md](../research/manufacturers/terraplex.md) for the brand-structure context)
- Model numbers = gallon capacity: i19 = 19 gallons, R32 = 32 gallons
