# Forge

> Dark dominant. Dramatic.

## When to use

Matches brand vibes: **Bold/Edgy**, **Modern/Tech-forward**. Good fit for dealers who want an intense, high-contrast aesthetic — glowing ember and flame motifs, dark-stone backgrounds, accent-color heat.

Currently used by: **Pyro Ag** (accent `#ea580c`, flame motif).

## Design specification

- Theme: dark backgrounds (stone-950, zinc-950), subtle section-to-section gradient shifts
- Hero: full-viewport video/image with animated glow blobs, gradient overlay tinted with accent color, ember accent lines at section edges. Use the shared `heroVideo` with `heroPoster` as fallback.
- Texture: radial gradient glows (large, blurred, low opacity), SVG noise/grain overlay, thin gradient border lines between sections
- Cards: dark elevated surfaces (stone-900/50) with backdrop-blur, border-hover glow in accent color, slight translate-y lift on hover
- Product layout: content + image side-by-side (R-32 image right, I-19 image left), spec cards as 2x2 grid below content with icon tiles
- Typography: font-bold to font-black headlines, tracking-tight, accent color gradient text on key words via bg-clip-text
