## Why this is library-worthy
A hero pattern that uses **only typography** — no photograph, no video, no illustration, no rendered object. The page opens with a single oversized display headline (a name, a thesis, a category) centered on an empty light surface, with a tiny inline stats grid directly below it (4 small label/value cells in a single row), and **then** a hero image appears below the type — but the image is the second moment, not the first. The structural inversion is the affordance: when the dealer's name / claim / specialty *is* the credibility move (a known operator, a known territory specialty, a known certification), the page should let the type carry the weight before any photograph competes for attention. Different from existing hero patterns: every other library hero leads with an image (photograph, video, render, illustration, or photographic mask). This pattern explicitly rejects an image-led opening — the type IS the hero.

## What to take
- Open the page with a centered single-column block, ~120px from the top of the viewport. No hero band — just empty negative space above and a single typographic moment.
- The display is oversized: ~96–160px depending on viewport, weight 700–900, tight letter-spacing (~ -1px to -3px), line-height 0.95–1.05. Two lines max — first line is one or two words, second line is one or two words. Centered horizontally. The display fills ~60% of the viewport width on desktop.
- The display content varies by use case:
  - **Profile / dealer landing**: the dealer name itself ("[DEALER NAME]" or first/last in stacked lines)
  - **Service landing**: the service name in declarative caps ("PART 137 / OPERATIONS")
  - **Product landing**: the product family ("R-32 / FLEET")
  - **About landing**: the dealer's positioning thesis in 2–4 words ("BUILT IN / IOWA")
- ~16–24px below the display, a small location/category line in muted gray (~#6B7280, 13–14px, regular weight, centered) — single short label ("United States" → for a dealer "[State] / Multi-county dealer").
- Below that, a tiny inline stats grid: 4 small cells in a single horizontal row, ~280–360px total width centered. Each cell is ~70–90px wide, separated by thin vertical hairlines. Per cell: large bold number on top (~22–24px), tiny all-caps label below (~9–10px). Stats are real and defensible — for the source they're "WORKS / 38 · SOTM / 3 · SOTD / 30 · OM / 37" (awards counts); for a dealer they're "YEARS / 12 · COUNTIES / 4 · DRONES / 23 · CERTS / 6" or similar.
- **Then** the hero image appears — a single landscape photograph (or constructed render) in a contained card with ~12–16px corner radius, occupying ~70% viewport width, centered, with vertical breathing room above (~80–120px from the stats grid). The image is large and dominant but it's the *second* moment, not the first. The eye goes type → location → stats → image, in that order.
- Below the image, a small flat 4-column metadata table (label/value rows: "LANGUAGES / English, French · STATUS / PROFESSIONAL · BUDGET / Over $50,000 · OPEN TO / Remote, In House"). Same density as a documentation page; same restraint as a dealer's license / certification block.
- The whole top of the page reads like a personal-brand profile or a documentation cover — restrained, type-led, immediately scannable.

## What NOT to take
- Don't add a CTA pill in the type block. The pattern's whole affordance is "the type is the hero" — a Get Started / Contact Us pill collapses the typographic moment into a generic "name + image + button" hero. The first interactive CTA appears below the image, in the body of the page.
- Don't substitute a logo for the display type. The pattern's credibility is *typed text* (a name, a thesis) — a logo collapses the moment into a brand-mark header, which is a different (and weaker) pattern.
- Don't add decorative elements around the type — no underline, no flanking rules, no kicker eyebrow. The display sits in pure negative space; chrome around it dilutes the affordance.
- Don't use this pattern when the dealer's name / thesis isn't the credibility move. If the dealer is new, unknown, or competes on price rather than reputation, the type-led opening collapses into a vanity title block. Use the [manifesto-creed hero](./refero-b86571f2-manifesto-creed-hero.md) (declarative + photo proof) or the [centered-screenshot-trio](./refero-f619e700-centered-screenshot-trio.md) (product-led light hero) instead.
- Don't fabricate stats. The 4-cell stats grid earns its space by being real and defensible — "YEARS / 12 · COUNTIES / 4 · DRONES / 23 · CERTS / 6" only works if those numbers are auditable. Generic "10K+ users / 99% satisfaction" stats collapse the documentation register into marketing chrome. If the dealer can't fill 4 defensible cells, drop the grid entirely and let the location line stand alone below the display.
- Don't crowd the type with a subhead paragraph. The display + tiny location line + 4-cell stats are the entire opening — adding a 2-line subhead paragraph between the display and the image collapses the type-led restraint. If supporting copy is needed, it goes *below* the image, not between display and image.
- Don't skip the image entirely. The image is the *second* moment but it IS necessary — without a hero image below the stats grid, the page reads as a profile-only page (which it is on the source) but for a dealer landing this collapses the affordance. Image earns its place by appearing after the type, not by competing with it.

## Adaptation notes (cross-archetype)
- TACTICAL: native fit. Display in tracked all-caps precision-sans (~96–120px, weight 800), monochrome black on white. Stats grid uses tabular-figure numerals (specifically — TACTICAL's typographic discipline). Hero image is a studio-shot product render or a controller close-up on seamless white. Add a 1px corner-bracket detail to the image card corners — consistent with the archetype's bracket motif. See [TACTICAL archetype](../../../archetypes/tactical.md). Pairs with the [spec-comparison-grid product-block](../../product-block/research/refero-713b66c0-spec-comparison-grid.md) below.
- CLEAN: native fit. Display in modern brand sans (~96–120px, weight 700), brand-charcoal on off-white (~#F8F8F6). Stats grid in muted gray with a single brand-navy accent on the numbers. Hero image is a clean equipment-in-context shot with soft drop shadow. Metadata table in brand-charcoal on white. See [CLEAN archetype](../../../archetypes/clean.md). Pairs with the [centered-screenshot-trio hero](./refero-f619e700-centered-screenshot-trio.md) on a sibling page (this hero is the dealer-name-led opener; the screenshot-trio is the product-led opener).
- TERRAIN: display weight stays heavy (~96–120px) but tracking opens up slightly to ~ -0.5px to evoke contour-line spacing. Stats grid uses a thin contour-line motif as the vertical hairlines between cells (a subtle topographic gesture at micro-scale). Hero image is a landscape band (aerial farmland or topographic terrain shot). After the image, the page drops to a "dark band" with topographic divider — the type-led opening is the page's first "light band." See [TERRAIN archetype](../../../archetypes/terrain.md). Sibling: [topographic layered-bg](./refero-ae84b62a-topographic-layered-bg.md) for an adjacent dark-band hero.
- FORGE: invert to a dark variant — display in white (~96–120px, weight 800) on near-black (~#0E0E0E) substrate. Stats grid in white on dark with a single ember-orange accent on the numbers. Location line in muted ember (~#FF8533, ~13px). Hero image below is a dusk / golden-hour equipment shot with rim-light. Metadata table in white-on-dark with thin charcoal hairlines. The dark substrate runs the whole page top — the image is a contained dark moment within the dark page register. See [FORGE archetype](../../../archetypes/forge.md). Pairs with the [hardware-lineup dark hero](./refero-0945a36c-hardware-lineup-dark-hero.md) on a sibling product page.
- FIELD: display in warm-display sans (~96–112px, weight 700), warm-charcoal on cream (~#F4EFE7). Stats grid in warm-charcoal with a single deep-amber accent on the numbers (matched to FIELD's restrained accent slot). Hero image below is a golden-hour operator-and-equipment shot. Metadata table in warm-charcoal on cream. The display can lean slightly more humanist than the geometric source — pair the display with a complementary serif if the dealer's brand has one. See [FIELD archetype](../../../archetypes/field.md). Pairs with the [editorial cream-bands product-block](../../product-block/research/refero-ecfaaa4f-editorial-cream-bands.md) below.
- PATRIOT: display in deep-navy heavy sans (~96–120px, weight 800) on cream substrate (~#F4EFE7). Stats grid in deep-navy with a single barn-red accent on ONE of the four numbers (the cell that matters most — typically counties served, or years operating). Location line in deep-navy. Hero image below is a late-afternoon American-farmland shot — operator with equipment, real region, no stock alpine landscapes. The single barn-red number is the page's only saturated moment in the opening — restraint is the move. Do NOT add stripe rules or star motifs around the display. See [PATRIOT archetype](../../../archetypes/patriot.md). Pairs with the [manifesto-creed hero](./refero-b86571f2-manifesto-creed-hero.md) on a sibling page where ideology leads.

## Source brief
Awwwards's [Aristide Benoist profile](https://www.awwwards.com/aristidebenoist/) — a designer-of-the-day profile page on the design awards site. The library-worthy move is the structural cadence (oversized display name centered → tiny location line → 4-cell stats grid → contained hero image below → flat metadata table), reusable for a Terraplex dealer when the dealer's *name / specialty / territorial credibility* is the credibility move and the page should let typography carry the opening before any photograph competes. Most directly transferable for a multi-county dealer with an operator reputation, a long-time Sam Welton-trained operator, or a regional specialist whose name on the door does the conversion work — a younger or unknown dealer should pick a different opener (see Pick guide below). The 4-cell stats grid maps cleanly to the dealer's auditable numbers (years operating, counties served, fleet size, certifications held).
