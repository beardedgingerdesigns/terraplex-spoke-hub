## Why this is library-worthy
A repeatable two-column product-block pattern: each feature gets its own full-width tinted stripe, alternating image-left / image-right between stripes, with the device or UI mockup nested inside a rounded card that matches the stripe color. Trades visual variety for systemic consistency — every block looks like a sibling of the next.

## What to take
- Each feature stripe is a full-bleed band with a distinct pastel fill (peach, butter yellow, lavender, etc.). Stripes stack vertically with no neutral spacers between — the color change IS the section break.
- Inside each stripe, a constrained two-column grid: text column (H2 + paragraph + secondary CTA) opposite a mockup column. The image/text sides swap on every other stripe so the rhythm reads ABABAB down the page.
- Mockups sit inside a rounded inset card (radius ~16–20px) the same color family as the stripe but a half-shade darker, giving the device a soft frame without a hard border.
- Section H2 is sentence-case sans-serif (~32–40px), paragraph stays short (2–3 lines), and the secondary CTA is a black pill with white label — same shape on every stripe so users learn the affordance once.
- Generous internal padding (~48px) and a consistent vertical rhythm between stripes (~64–80px) keep the alternating stripes from feeling busy.

## What NOT to take
- The cheerful pastel palette (peach #FFEDE6, butter #FFF7D8, lavender #E7D6FF) is the source-screen vibe — borrowing it wholesale would violate FORGE/TACTICAL's dark-dominant rule.
- The display serif headline above the hero is from a different pattern (see hero) — don't conflate. This note is about the product-block stripes only.
- Don't import the alternating rhythm if there are fewer than three feature blocks — with two, AB just looks like an oversight.

## Adaptation notes (cross-archetype)
- CLEAN: native fit — use brand-tinted very-pale neutrals (e.g. #F4F1EC, #EFF2F0) instead of saturated pastels to keep the corporate-professional read.
- TERRAIN: this IS the topographic alternating-bands pattern at section scale — alternate light cream and a deeper sage / clay tone instead of two pastels, and let the stripe transition itself act as the topographic line.
- FIELD: swap pastels for muted earth tones (warm cream, soft amber, sage, dusty terracotta). Keep the two-column grid and pill CTA shape; replace SaaS screenshots with photographic crop or equipment imagery framed in the same rounded inset.
- PATRIOT: stripes are already core to this archetype, so the ABABAB rhythm fits natively — but rotate three colors instead of two (cream → navy → cream → barn-red → cream → navy) to avoid reading as a literal flag. Keep the rounded mockup-card inset; thin star or diagonal-rule divider between stripes only if the rest of the page is restrained. The pill CTA stays a single neutral (charcoal) so the flag colors don't compete for emphasis.
- FORGE / TACTICAL: the alternating-pastel-stripe move directly conflicts with dark dominant; if you want this rhythm in a dark archetype, rebuild as alternating black / charcoal stripes with a single accent-color rim — a different pattern, not this one.

## Source brief
GlossGenius — a custom-website-builder landing page for salon owners. The dealer brief was conveying multiple product capabilities (booking, SEO, customization) without a single dense feature grid; the alternating pastel stripes give each capability its own full-width room while keeping the page systemic.
